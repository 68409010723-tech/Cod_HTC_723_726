HTML Attributes

<a>

แท็ก <a> ใช้สำหรับสร้างไฮเปอร์ลิงก์ แอตทริบิวต์ href ระบุ URL ของหน้าเว็บที่ลิงก์นั้นนำไป
ตัวอย่างโค๊ด <a href="https://www.w3schools.com">Visit W3Schools</a>

<img>

แท็ก <img> ใช้สำหรับฝังรูปภาพลงในหน้า HTML โดยแอตทริบิวต์ src จะระบุพาธไปยังรูปภาพที่จะแสดง
ตัวอย่างโค๊ด <img src="img_girl.jpg">
