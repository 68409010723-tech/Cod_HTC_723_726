HTML Basic

<!DOCTYPE>

การประกาศ <!DOCTYPE> แสดงถึงประเภทของเอกสาร และช่วยให้เบราว์เซอร์แสดงผลเว็บเพจได้อย่างถูกต้องต้องปรากฏเพียงครั้งเดียวเท่านั้น ที่ด้านบนสุดของหน้า (ก่อนแท็ก HTML ใดๆ)
การประกาศ <!DOCTYPE> ไม่คำนึงถึงตัวพิมพ์ใหญ่เล็ก
การประกาศ <!DOCTYPE> สำหรับ HTML5: ตัวอย่าง <!DOCTYPE html>

<html>

เอกสาร HTML เริ่มต้นด้วย <html> และจบด้วย </html>

<body>

ส่วนที่มองเห็นได้ของเอกสาร HTML อยู่ระหว่าง <body> และ </body>

<h1 - h6>

หัวข้อใน HTML กำหนดด้วยแท็ก <h1> ถึง <h6> ซึ่ง <h1> กำหนดหัวข้อที่สำคัญที่สุด <h6> กำหนดหัวข้อที่สำคัญน้อยที่สุด

<p>

ย่อหน้าใน HTML กำหนดด้วยแท็ก <p>
